# 0x00. Shell, basics

This directory contains Bash scripts that demonstrate fundamental shell commands and navigation in Linux/Unix systems.

## Description

This project is part of the System Engineering & DevOps curriculum. It focuses on understanding the basics of shell scripting, file system navigation, and essential Linux commands.

## Learning Objectives

By completing this project, you will learn:
- What is the shell and what is the terminal
- How to navigate the file system
- How to manipulate files and directories
- Working with commands like `pwd`, `ls`, `cd`, `cp`, `mv`, `rm`, `mkdir`, etc.
- Understanding absolute and relative paths
- File permissions and how to change them

## Requirements

- All scripts are tested on Ubuntu 20.04 LTS
- All scripts are exactly two lines long (`wc -l file` should print 2)
- All files must start with `#!/bin/bash`
- All files must be executable
- You are not allowed to use backticks, `&&`, `||` or `;`

## Files

| Filename | Description |
|----------|-------------|
| `0-current_working_directory` | Prints the absolute path name of the current working directory |

## Usage

To run any script:

1. Make the script executable:
```bash
chmod +x script_name
```

2. Execute the script:
```bash
./script_name
```
